# Steam Game Recommender System

A collaborative filtering recommender system built using Apache Spark MLlib on the Steam dataset.

## Overview
Steam game recommender system using ALS matrix factorisation

## Technologies Used
- Apache Spark (PySpark + Spark SQL)
- Databricks
- MLlib (ALS Collaborative Filtering)
- MLflow (Experiment Tracking)

## Dataset
Steam 200k dataset (available on Kaggle)

## Results
- Best ALS model achieved RMSE of 1.4620 (rank=30, regParam=0.1)
- Matrix sparsity of 99.83% handled effectively through collaborative filtering
